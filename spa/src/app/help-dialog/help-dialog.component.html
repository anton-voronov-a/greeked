Enable general extension support setting in Nightly.

Tap on the three dot menu and select Settings.

Tap on About Firefox Nightly.

Tap the Firefox Nightly logo five times until the “Debug menu enabled” notification appears.

Navigate back to Settings. You will now see a new entry for “Custom Add-on collection”. Once a custom add-on collection has been set up, this menu item will always be visible.

Tap the “Custom Add-on collection” and set the value "16956994" for the User ID and "My" for the collection name.
